Sol-37 Site README

This site is a Windows-95-style shell wrapped around a merged public archive. It now functions as desktop, file browser, blog reader, star-map program, traffic monitor, and IRC-backed public logbook. The point is no longer just mood; it is a navigable operating environment for the material living under random/www.

What The Visitor Sees

  • Desktop shell with draggable Win95 windows, taskbar buttons, Start menu, and iframe-backed program windows.
  • Blog for Markdown material listed from posts/index.json.
  • HTML archive pages mounted as document windows and desktop icons.
  • Site Map rebuilt as an Explorer-like browser with address bar, search, folders pane, and working menus.
  • Star Map as a proper interactive program window that can also pop out into a real browser tab.
  • Public Logbook as an IRC-style chat client backed by a real local IRC server.

Archive Layout

  • index.html is the retro shell and primary landing page.
  • posts/ contains Markdown and HTML content surfaced through Blog, Start menu, and desktop icons.
  • programs/ contains full app-like pages such as the Star Map and Logbook.
  • Top-level HTML files remain directly reachable and are now cataloged by the Site Map.
  • Media files, zips, images, JSON, and utility artifacts still live alongside the archive instead of being hidden away.

Programs And Systems

  • Site Map opens files in separate retro windows and lives on the taskbar as a map button.
  • Traffic Monitor opens above the clock and derives hit counts from local access logs.
  • Public Logbook supports named posting into #public-logbook, #archive-watch, and #civilization-sim.
  • IRC backend is ngircd on port 6667, with a local logger bot preserving channel history for the web client.
  • HexChat is configured locally as a real desktop IRC client against the same backend.

Networking And Hosting

  • The site itself is served locally by Caddy on 127.0.0.1:8888.
  • sol.system42.one is published through a named Cloudflare tunnel.
  • irc.system42.one currently still points at the tunnel path; the direct-public IRC cutover was deferred.
  • The IRC daemon is now listening on both IPv4 and IPv6 locally, so the machine is ready once DNS/NAT choices are finalized.

Terminal Commands

  • help, about, date, ls, theme ...
  • posts, openpost <id>, cat <id>
  • sitemap opens the file browser
  • star opens the Star Map
  • logbook opens the IRC-backed public logbook
